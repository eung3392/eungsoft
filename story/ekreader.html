스마트 글래스에서 즐기는 편안한 독서, EKReader 출시

안녕하세요! EUNG SOFT입니다. Rokid Glasses 기기에서 더욱 편리하게 책을 읽을 수 있도록 도와주는 e-북 리더 애플리케이션, EKReader를 소개합니다.

EKReader는 텍스트(TXT) 및 이펍(EPUB) 파일을 지원하는 깔끔하고 심플한 e-북 리더 앱입니다. Rokid Glasses에서 눈의 피로를 덜고 조작이 쉽도록 다크 모드 기반의 블랙 테마와 심플한 인터페이스를 구성하는 데 중점을 두었습니다.

주요 기능 및 화면 구성

  • 메인 및 리더 화면: 기기 내의 도서 목록을 리스트 형태로 보여주며 파일 형식을 아이콘으로 구분합니다. 리더 화면에서는 하단에 실시간으로 독서 진행률(%)과 프로그레스 바를 표시하여 내가 얼마나 읽었는지 시각적으로 확인할 수 있습니다.
  • 페이지 이동: 5% 단위로 위치를 조절해 빠르게 넘어갈 수 있습니다.
  • 글자 크기 조절: 가독성에 맞춰 Small, Normal, Large, X-Large 4단계로 폰트 크기를 쉽게 변경할 수 있습니다.

Web Install 기기 파일 전송 연동

EUNG SOFT 홈페이지의 Web Install 메뉴를 이용하면, 로키드 글래스를 연결한 후 복잡한 과정 없이 ‘EKReader 전용 파일 올리기’ 버튼 하나로 곧바로 각종 도서 자료(txt, epub)를 글래스 내부 전용 폴더로 손쉽게 전송할 수 있습니다.

복잡한 버튼을 최소화하고 꼭 필요한 기능에만 집중할 수 있도록 개발한 EKReader와 함께, 언제 어디서든 즐거운 독서 시간을 가져보시길 바랍니다!

개선할 점

만들면서 페이징 처리하는 부분이 제일 신경쓰였습니다. 아직 페이징 처리가 미흡해서 하단 영역이 폰트의 크기에 따라 잘려 보이는 현상이 있을 수 있습니다. 나름 조정한다고 하기는 했습니다. 현재 보고 있는 화면에서 폰트 사이즈를 변경 했을 때에 보고 있던 정확한 화면으로 이동하는 부분도 아직은 어색합니다. 얼추 비슷한 곳으로 이동하는 수준입니다. 그리고, 좀 답답할 수도 있겠지만 용량이 큰 파일을 읽을 때에는 로딩 시간이 걸립니다. 조금 기다려주시면 나타날 겁니다. 이 부분도 많이 아쉽더라구요. 조금씩 개선할 방안을 찾아가면서 업데이트 하도록 하겠습니다. 감사합니다.

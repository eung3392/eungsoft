프론트엔드 개발을 위한 Javascript & Vue.js 핸드북

BOOKK(부크크) | 2022.08.30

이 책은 처음 시작하는 분들을 위한 입문서라기보다는 기본 지식이 있는 상태에서 포트폴리오를 만들거나, 업무를 진행하는 분들을 위한 코드 참고 서적이다. 일을 하다 보면 특정 문법이 생각이 나지 않는 경우가 있는데 그럴 때 참고했으면 해서 정리한 책이기도 하다. 그래서 최대한 책을 작게 만들었다. 설명도 많이 없는 편이다. 간결한 샘플 코드 위주로 정리된 책이 필요하신 분이라면 좋을 듯하다. 이 책에서는 기본적으로 Javascript 문법을 다루면서 ES6 문법을 비교해서 볼 수 있도록 구성하였다. Vue.js의 2, 3 버전은 장을 따로 분리하였고 3 버전에서 옵션 API 와 컴포지션 API 방식의 코드를 비교해서 작성하였다. 이 책이 Vue.js 와 프론트엔드 개발에 관심을 가지는 분들께 도움이 되었으면 좋겠다.

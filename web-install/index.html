usb Web Install (APK)

별도의 프로그램 설치 없이, 브라우저에서 바로 APK를 장치에 설치합니다.

install_mobile Web Install Count: ...
warning

브라우저 제한 안내

WebUSB API는 Chrome, Edge, Opera 등 Chromium 기반 브라우저에서만 작동합니다. Safari나 Firefox에서는 이 기능을 이용할 수 없습니다.

기기 연결 안 됨

menu_book 사용법 및 준비사항

  • looks_one 안드로이드 기기의 개발자 옵션에서 USB 디버깅을 활성화합니다.
  • looks_two USB 케이블로 기기와 PC를 연결합니다. (데이터 통신 가능 케이블 필수)
  • looks_3 "기기 연결하기" 버튼을 클릭하고, 브라우저 팝업에서 기기를 선택합니다.
  • looks_4 기기 화면에 "USB 디버깅을 허용하시겠습니까?" 팝업이 뜨면 허용을 누릅니다.
  • looks_5 연결 성공 후 APK 파일 선택 및 설치 버튼으로 원하는 APK를 선택합니다.

fact_check 점검 사항 (문제가 발생할 경우)

  • check_circle 브라우저 확인: Chrome(크롬)이나 Edge(엣지)를 사용 중이신가요? (Safari, Firefox는 지원 안 함)
  • check_circle USB 디버깅: 기기(안드로이드/로키드 등) 설정에서 USB 디버깅이 켜져 있나요?
  • check_circle 케이블 연결: 기기가 PC와 USB로 직접 연결되어 있나요? (충전 전용 케이블이 아닌 데이터 통신용 필수)
  • warning ADB 서버 충돌: 터미널에서 adb kill-server를 실행한 후 다시 연결하세요. (ADB 서버와 WebUSB는 동시 사용 불가)
